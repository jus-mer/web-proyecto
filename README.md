# web-proyecto

Este repositorio aloja la página web del Proyecto FONDECYT Regular "Justicia de mercado y Merecimiento del bienestar".

La página fue construida en casi su totalidad con Quarto, y tomó como inspiración el sitio de [ALA Labs](https://labs.ala.org.au/)

para acceder a la página, haga click [aquí](https://tomasurzuam.github.io/web-proyecto/docs/).
